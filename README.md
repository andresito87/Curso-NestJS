## Perfil del proyecto

Este repositorio reúne los ejercicios, ejemplos y apuntes del curso con una intención muy concreta: convertir conceptos de backend en código legible, modular y fácil de mantener.

La línea de trabajo se apoya en TypeScript como base, NestJS como framework principal y una serie de prácticas que ayudan a pensar aplicaciones con estructura real de producción.

## Lo que trabajamos en este repositorio

- dominio progresivo de TypeScript;
- comprensión de clases, interfaces, tipos, objetos y decoradores;
- aplicación de inyección de dependencias y separación de responsabilidades;
- consumo de APIs externas con adaptadores reutilizables;
- uso de estructuras pensadas para escalar en complejidad sin perder claridad.

## Stack y herramientas

- **Node.js**: runtime del entorno de desarrollo.
- **TypeScript**: lenguaje principal para tipado estático y modelado.
- **NestJS**: framework backend orientado a arquitectura modular.
- **Vite**: soporte de desarrollo para los ejercicios base.
- **Axios**: cliente HTTP para integración con APIs externas.
- **Prisma**: ORM para manejo de bases de datos relacionales.
- **MongoDB**: base de datos NoSQL para ejemplos de persistencia flexible.
- **PostgreSQL**: base de datos relacional para ejemplos de consultas complejas.
- **Postman**: herramienta para probar y documentar APIs REST.
- **HTML y CSS**: capa visual mínima para los ejemplos de navegación.
- **Git**: control de versiones del material del curso.

## Contenido del curso

- fundamentos de TypeScript;
- clases, objetos e interfaces;
- inyección de dependencias;
- decoradores;
- módulos, controladores, providers y servicios;
- pipes, guards, interceptors y filters;
- validación y transformación de datos;
- autenticación y autorización;
- consumo de bases de datos relacionales y NoSQL;
- pruebas unitarias y pruebas end-to-end;
- documentación de APIs y despliegue.

## Cómo ejecutarlo

Desde la carpeta correspondiente a cada parte, ejecuta:

```bash
npm install
npm run dev
```

Otras tareas útiles:

```bash
npm run build
npm run preview
```
